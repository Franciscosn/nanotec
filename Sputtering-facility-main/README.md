# Sputtering Facility

## Start (einfach)

Im Projektordner:

```bash
python run.py
```

Alternativ per Doppelklick:

- macOS: `Start_Sputtering.command`
- Windows: `Start_Sputtering.bat`

## Runtime-Settings

Beim Start wird automatisch `sputter_settings.json` geladen (falls vorhanden).
In der GUI kannst du Modus (Simulation/Real), Pfeiffer-Backend und Ports direkt umstellen,
aus Datei laden/speichern und den Controller neu starten.
